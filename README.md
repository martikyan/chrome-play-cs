# chrome-play-cs

A Chrome browser extension, that adds features to the game in at play-cs.com

## Features
- Better pointer with customization
- Kill sound. For that, you'll need to log in.
- Shortcuts to buy weapons quickly
- Display settings
- Custom player models
- Jump fix using space on some platforms

Credits to [DHunterZ on downloading-cs.su](https://downloading-cs.su/) for the custom models.

---

## How to Install

Since this extension is currently not on the Chrome Web Store, you will need to install it manually using developer mode:

1. **Download the latest release:**
   Go to the [Releases page](https://github.com/martikyan/chrome-play-cs/releases) and download the latest version (usually as a `.zip` file).
2. **Extract the files:**
   Unzip the downloaded folder somewhere safe on your computer.
3. **Open Chrome Extensions:**
   In your Chrome browser, navigate to `chrome://extensions/`.
4. **Enable Developer Mode:**
   Toggle the **Developer mode** switch in the top-right corner of the page to **ON**.
5. **Load the extension:**
   Click the **Load unpacked** button in the top-left corner, select the extracted folder (the folder containing the `manifest.json` file), and click **Open**.

## How to Update

To get the latest features and bug fixes, you will need to update the extension manually:

1. **Download the new version:**
   Check the [Releases page](https://github.com/martikyan/chrome-play-cs/releases) and download the latest release `.zip`.
2. **Replace the files:**
   Extract the new files, replacing the contents of your old extension folder.
3. **Reload the extension:**
   Go to `chrome://extensions/` and click the **Update** button at the top, or click the **Reload** (circular arrow) icon on the **chrome-play-cs** card to apply the changes.
4. **Verify the update:**
   Check the version number shown on your `chrome://extensions/` page for **chrome-play-cs**. 
   > ⚠️ **Note:** Always make sure the version displayed in Chrome matches the version of the release you just downloaded from the [Releases page](https://github.com/martikyan/chrome-play-cs/releases) to confirm the update applied successfully.
